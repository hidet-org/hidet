Installation
============

Run the following command to install ``hidet`` package via python pip:

.. code-block:: console

  $ pip install hidet

To verify the installation, run the following command:

.. code-block:: console

  $ python -c "import hidet"

.. tip::

  You can also install the nightly version of ``hidet`` package via python pip:

  .. code-block:: console

    $ pip install --pre --extra-index-url https://download.hidet.org/whl hidet


If you want, you can also :doc:`build from source </getting-started/build-from-source>`.
